# AlmacenGT Theme — Project Index

> WordPress + WooCommerce theme for a Guatemalan online marketplace.
> Design goal: replicate the layout and feel of **BestBuy.com**.

---

## Stack
- WordPress + WooCommerce (standard PHP, no headless)
- Pure PHP templates — no Node.js, no build process
- CSS3 with custom properties, no preprocessor
- Vanilla JS (jQuery available via WordPress)

## Theme Location
`d:\Custom Websites\AlmacenGT\almacengt-theme\`
- Text domain: `almacengt`
- Asset version: `2.4` (increment in `functions.php` on every CSS/JS change)
- Container max-width: `1120px`
